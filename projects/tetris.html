Overview

This project was developed as a collaborative JavaFX game application. It includes core Tetris gameplay such as block movement, rotation, collision detection, line clearing, scoring, and game state management.

Key Challenges

  • Managing source code and version control through Git/GitHub in a team environment.
  • Refactoring the project structure to apply MVC architecture.
  • Separating game logic, UI rendering, and keyboard input handling.
  • Maintaining readable and reusable code while supporting both single-player and two-player modes.
