HealthScan
App

A Unity-based smart retail mobile application that helps users scan food products, review nutrition information, save favourites, and receive healthier product recommendations through API and Firebase integration.

View Demo ↓ Back to Selected Work

Overview

HealthScan was designed as a mobile retail experience for quick product lookup and healthier decision-making. The app combines barcode scanning, OpenFoodFacts product data, Firebase services, and mobile-first UI screens built in Unity.

Key Features

  • Barcode scanning flow for retrieving packaged food information.
  • OpenFoodFacts API integration for nutrition and product details.
  • Firebase-backed user data flows for saved products and app state.
  • Product recommendation screens focused on healthier alternatives.
  • Responsive mobile UI layouts designed for Android and iOS behaviour.

Key Challenges

  • Handling camera behaviour consistently across mobile devices.
  • Designing UI layouts that scaled cleanly across screen sizes.
  • Connecting API data, Firebase state, and Unity UI updates smoothly.
  • Balancing functional product lookup with a simple shopping-friendly flow.

Demo

The demo shows the app flow from scanning and browsing product information through to recommendation-oriented screens.

Tech Stack

Unity · C# · Firebase · OpenFoodFacts API · Mobile UI/UX

My Role

Frontend design and implementation, API integration support, mobile UI flow, and feature testing across product scanning screens.

Focus Area

Mobile interaction design, camera-driven workflows, API data handling, Firebase integration, and user-friendly retail experiences.

Outcome

Delivered a working prototype that demonstrates product scanning, nutrition lookup, saved data flows, and recommendation features.
